## 0.1.0

* Initial release of text_avatars package
* UserAvatar widget with support for:
  - Network image display with loading indicators
  - Automatic initials generation from user names
  - Custom text display
  - Smart fallback to initials on image load failure
  - Consistent color generation based on initials
  - Customizable size, colors, and text styles
  - Automatic text contrast for readability
  - Material Design color palette
