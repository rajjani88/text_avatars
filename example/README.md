# text_avatars_example

Demonstrates how to use the text_avatars package.

## Getting Started

This example app showcases all features of the text_avatars package including:

- Basic initials generation from names
- Custom text display
- Custom colors and styling
- Different avatar sizes
- Network image loading with fallback
- Usage in lists

## Running the example

```bash
cd example
flutter pub get
flutter run
```

## Features Demonstrated

1. **Basic Initials** - Auto-generated initials from user names
2. **Custom Text** - Display custom text or numbers
3. **Custom Colors** - Background and text color customization
4. **Different Sizes** - Avatars in various sizes (32px to 120px)
5. **Network Images** - Loading images from URLs with automatic fallback
6. **Custom Text Styles** - Font weight and size customization
7. **List Integration** - Avatars used in a typical user list
